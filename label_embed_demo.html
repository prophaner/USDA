USDA Label Embed Demo

This page demonstrates how to embed a USDA Nutrition Label in your web application.

How to Use

To embed a USDA Nutrition Label in your application:

  1. Make a POST request to the /label/ endpoint with your recipe data
  2. Get the embedded_html from the response
  3. Insert the HTML code into your page

Example Code

// JavaScript example
fetch('http://127.0.0.1:8000/label/', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        recipe_title: "Chicken Salad",
        recipe_data: { /* Your recipe data */ },
        business_info: { /* Your business info */ },
        label_type: "USDA (Old FDA) Vertical"
    })
})
.then(response => response.json())
.then(data => {
    // Display the embedded label
    document.getElementById('label-container').innerHTML = data.embedded_html;
    
    // Set up download links
    document.getElementById('pdf-link').href = data.pdf_download_url;
    document.getElementById('png-link').href = data.png_download_url;
})
.catch(error => console.error('Error:', error));

For more information, refer to the API documentation.
